## Talk Track Standards

Read the full style guide: library/creative-dna/writing/talk-track-style-guide.md

### Quick Reference
- Voice: Conversational, vulnerable, personal — "a friend sharing something important"
- Structure: Hook → Framework → Activity → Demo → Climax → Closing
- Opening: Start with a personal story or question that creates connection
- Delivery: Short sentences. Pauses marked with [PAUSE]. Natural rhythm.
- Vulnerability: Share real struggles and doubts — this builds trust
- Emotional arc: Build from curiosity → understanding → personal → transformative

### Formatting
- Speaker notes in **bold** for emphasis cues
- [PAUSE] for deliberate pauses
- [CLICK] for slide build triggers
- [TABLE ACTIVITY] for group exercises
- Approximate timing for each section

### Slide Integration
Reference the slide being discussed: "On this slide, you see..."
Build sequences align with talk track sections — each [CLICK] reveals the next point.

### Versioning
- Save as {topic}-talk-track-v1.md, v2.md, etc.
- TTS script (stripped of stage directions) as {topic}-tts-script.md
